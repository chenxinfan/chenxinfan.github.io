Go语言入门教程

Go语言是由Google开发的一种静态类型、编译型语言，具有简洁的语法和强大的并发特性。

安装Go

首先需要下载并安装Go语言环境：

# 下载Go
wget https://golang.org/dl/go1.21.0.linux-amd64.tar.gz

# 解压到/usr/local
sudo tar -C /usr/local -xzf go1.21.0.linux-amd64.tar.gz

# 设置环境变量
export PATH=$PATH:/usr/local/go/bin

第一个Go程序

创建一个简单的Hello World程序：

package main

import "fmt"

func main() {
    fmt.Println("Hello, Go!")
}

基本语法

变量声明

// 方式1：var关键字
var name string = "Go"

// 方式2：短变量声明
age := 25

// 方式3：类型推断
var city = "Beijing"

函数定义

func add(a, b int) int {
    return a + b
}

// 多返回值
func divide(a, b int) (int, error) {
    if b == 0 {
        return 0, fmt.Errorf("除数不能为零")
    }
    return a / b, nil
}

并发编程

Go的goroutine让并发编程变得简单：

func main() {
    go func() {
        fmt.Println("异步执行")
    }()
    
    time.Sleep(time.Second)
}

总结

Go语言具有以下特点：

  • 简洁的语法
  • 强大的并发支持
  • 快速的编译速度
  • 丰富的标准库
  • 跨平台支持

学习Go语言是提升编程技能的好选择！
